# Hezpo E-commerce SEO Skill

Use this skill when the user asks for product page copy, blog structure, SEO content, Shopee/TikTok support content, A+ image text, or conversion-oriented website improvements.

## Business goal

Support Hezpo's e-commerce growth by turning verified product and sales context into clear, searchable, conversion-focused content.

Priority outcomes:

- More product inquiries
- Better product education
- Better Malaysia-market SEO coverage
- Stronger Shopee/TikTok support pages
- Better B2C / wholesale / dealer / salon routing
- Content that sales staff and Codex can reuse

## Priority products

Use verified project/user context first. Known priority products include:

- TOPPIK Hair Building Fiber Black
- Hermoso Professional Extra Hold Hair Spray 420ml
- HSIS / HSSIS Hair Clay
- ALKEN shampoo / masque ranges
- Carexpert Argan Oil
- HEADS Hairloss Series
- V&G V-1248 Hair Straightener

## Malaysia-market buyer angles

Use practical buyer pain points:

- Humid weather
- Sweat
- Helmet hair
- Hair collapsing after styling
- Frizz and dryness
- Flat roots
- Salon retail trust
- Bulk reseller margin
- Product education for first-time buyers

## Content rules

- Do not fabricate ingredients, certifications, awards, clinical results, or before/after guarantees.
- Avoid medical or guaranteed hair-growth claims unless verified product documentation is provided.
- Do not copy competitor text.
- Keep language simple and sales-useful.
- When requested, create Chinese / English / Malay versions.
- For website copy, prefer short sections that can be reused in cards, product pages, FAQs, and blog posts.
- For Shopee/TikTok support copy, prioritize hooks, buyer objections, demo scenes, CTA, and repeatable short-video ideas.

## Recommended output formats

For product pages:

1. Hero headline
2. Short subheadline
3. Key selling points
4. Target customer
5. How to use
6. FAQ
7. Wholesale/dealer/salon angle
8. SEO title/meta description

For blog/SEO:

1. Keyword target
2. Search intent
3. Outline
4. Product mention points
5. FAQ schema candidates
6. Internal links

For short-video content:

1. Hook
2. Scene
3. Problem
4. Product action
5. Proof/demo angle
6. CTA
7. Hashtags

## Validation

If editing website code or content files, run:

```bash
npm run build
```

If only producing copy or a markdown report, no build is required unless repo files were changed.
